## parcel 1.9.4 已经内置vue的支持,并且会自动安装项目依赖

`parcel`一个快速，零配置的 Web 应用程序打包工具,这里我介绍下如何和`vue`结合进行开发。具体代码:https://github.com/zlxbuzz/parcel-demo

## Getting started

1. clone:

```shell
git clone https://github.com/zlxbuzz/parcel-demo.git
```


2. Install with yarn:

```shell
cd parcel-demo
yarn
```

3. dev or build:

```shell
yarn dev
yarn build
```
